# Prerequisites

Before installing OpenShift, ensure your environment meets the requirements outlined in this section.

## General Requirements

- A Red Hat account with an active OpenShift subscription or evaluation
- Access to [console.redhat.com](https://console.redhat.com) to download the installer and pull secret
- DNS and networking configured for your target environment
- Sufficient compute, memory, and storage resources

## Tools

- `openshift-install` CLI
- `oc` CLI
- `kubectl` (optional, bundled with `oc`)

!!! tip
    Download the latest installer and CLI tools from the [Red Hat Hybrid Cloud Console](https://console.redhat.com/openshift/downloads).
