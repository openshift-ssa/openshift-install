# Red Hat Enterprise Linux Install

[Back to Install](index.md#build-the-bastion-host)

Follow the RHEL 9.x installation wizard, selecting **Server with GUI** as the software selection.

The key steps during installation:

1. Boot the host from the RHEL 9.x Binary DVD ISO
2. Select your language and keyboard layout
3. Configure the installation destination (select the target disk)
4. Set the software selection to **Server with GUI**
5. Configure the network and hostname
6. Set the root password and create an administrative user
7. Enable SSH access for the user
8. Begin the installation and reboot when complete
