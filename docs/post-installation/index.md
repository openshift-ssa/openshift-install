# Post-Installation

After a successful OpenShift installation, there are several configuration steps to complete.

## Common Tasks

- Configure identity providers (LDAP, OIDC, HTPasswd, etc.)
- Set up persistent storage (ODF, NFS, cloud-native CSI drivers)
- Install Operators from OperatorHub
- Configure ingress and certificates
- Set up cluster logging and monitoring
- Apply RBAC policies and project quotas
- Configure image registry storage
